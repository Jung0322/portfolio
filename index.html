01 — About

저를 소개합니다

4년차 웹 개발자로, 다양한 고객사의 비즈니스 요구사항을 분석하여 실제 업무 시스템으로 구현하는 역할을 수행해 왔습니다. 단순 개발을 넘어 사용자의 업무 흐름을 파악해 화면 정의서를 작성하고, 이에 최적화된 DB 모델링을 직접 진행하며 프론트엔드와 백엔드를 아우르는 풀스택 개발자로 성장했습니다.

Oracle과 MSSQL을 활용한 데이터 설계부터 구매포탈, HR, SAP 등 사내 주요 시스템과의 인터페이스 연동 업무를 안정적으로 처리했습니다. 또한 SVN, Git을 통한 형상관리와 Jenkins 기반의 빌드/배포 자동화 환경을 구축하여, 복잡한 비즈니스 로직이 포함된 서비스도 효율적으로 운영할 수 있는 실무 역량을 보유하고 있습니다.

"설계부터 배포까지, 비즈니스 로직을 안정적인 코드로 구현"
Vue SpringBoot Jenkins Gitea Nginx Oracle/MSSQL

Full-Stack 개발

Vue와 Spring Boot를 주력으로, 프론트엔드부터 백엔드까지 유기적으로 연결된 서비스를 구현합니다.

DB 및 화면 설계

고객 인터뷰를 바탕으로 화면 정의서를 작성하고, Oracle/MSSQL 기반의 최적화된 DB를 설계합니다.

빌드/배포 자동화

Jenkins를 활용하여 소스 코드의 빌드부터 배포까지의 과정을 자동화하여 운영 효율을 극대화합니다.

형상관리

Git과 SVN을 병행 활용하여 소스 이력을 정밀하게 관리하고, 협업 시 충돌 없는 환경을 유지합니다.
